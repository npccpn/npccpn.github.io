看就完事了！

为自己的hexo博客添加RSS

  1. 在博客根目录下安装RSS插件

    npm i hexo-generator-feed

  2. 在根目录下的_config.yml文件中找到Extenxions进行如下配置

    # Extensions
## Plugins: https://hexo.io/plugins/
plugin:
  hexo-generator-feed
# Feed Atom
feed:
  enable: true
  type: atom
  path: atom.xml
  limit: 20

  3. 在主题配置的config文件中开启RSS页面符号

    rss: /atom.xml

  4. 效果图
